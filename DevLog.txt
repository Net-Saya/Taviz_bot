Update 26.03.26 1.2
AUTO REPORT SYSTEM & ENHANCED STATISTICS
-Додана команда /startrecord для активації автоматичної відправки звітів
-Додана команда /stoprecord для деактивації автоматичної отправки
-Реалізована система автоматичної відправки Excel файлів кожного місяця (1-го числа о 9:00)
-Excel звіти відправляються ТІЛЬКИ в особисту розмову адміністратору, який активував /startrecord
-Конфігурація зберігається у JSON файлі (auto_report_config.json) для збереження при перезагрузці бота
-Додана залежність APScheduler для планування автоматичних задач
-Система перевіряє права адміністратора при активації /startrecord
-Звіт містить статистику повідомлень за попередній місяць

STATISTICS IMPROVEMENT
-Переведено на filters.ALL для обробки всіх типів повідомлень (раніше тільки текст)
-Добавлен розділений облік: текст, фото, стикери, GIF для кожного користувача
-Система корректно розрізняє document із mime-type: image/* (як фото) та video/mp4 (як GIF)
-Додано фільтрування команд (/startrecord, /stoprecord тощо) від підрахунку статистики
-Реалізовано обробку caption в медіа-файлах для логування

LOGGING & USER INTERFACE
-Подробне логування типів сообщений: [text], [photo], [sticker], [gif], [unknown]
-Логи показують тип контенту и текст/caption якщо він присутній
-Команди тепер перехоплюються и не вважаються за статистику

EXCEL FORMATTING & DESIGN
-Створена централізована функція create_stats_workbook для та табліц
-Форматування Excel: синя шапка з білим текстом (колір #4F81BD)
-Додано обрамлення всіх ячеек (thin border)
-Центровка даних в числових колонках
-Оптимізовані ширини колонок: Користувач (30), Текст (12), Фото (12), Стикери (12), GIF (12), Загалом (14)
-Єдиний стиль для всіх типів генерації отчетів (автоматичні, на запит)
